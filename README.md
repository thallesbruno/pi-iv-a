# pi-iv-a

Repositório de artefatos do **Projeto Integrador IV-A** do curso de **Análise e Desenvolvimento de Sistemas** da **PUC Goiás**.

---

## Sobre o Projeto

A aplicação consiste em um projeto desenvolvido em **Java**, com foco em:

- Implementação de funcionalidades (*features*) voltadas a uma organização;
- Controle de versionamento do código-fonte;
- Aplicação de testes unitários utilizando a biblioteca **JUnit**.

---

## Tecnologias Utilizadas

- **Java**
- **JUnit**
- **Git**
- **GitHub**

---

## Objetivos

Este projeto tem como objetivo:

1. Praticar o desenvolvimento de aplicações Java;
2. Aplicar conceitos de versionamento de software;
3. Desenvolver testes automatizados;
4. Utilizar boas práticas de organização e manutenção de código.
